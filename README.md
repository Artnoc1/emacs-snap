# GNU Emacs #

-------------------------------------------------------------------------------

**GNU Emacs in a snap** - the extensible, customizable, self-documenting real-time display editor

## Installation ##

``` shell
sudo snap install emacs --classic
```

([Don't have snapd installed?](https://snapcraft.io/docs/core/install))
